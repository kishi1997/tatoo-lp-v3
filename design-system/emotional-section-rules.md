# emotional-section-rules.md
# Problem系

## 対象セクション

- Problem Section
- Challenges of Self-Learning Section
- 不安訴求 / 共感訴求 / 警鐘系セクション

## 目的

未経験者や独学者が抱える不安を言語化し、次の解決策セクションへ自然に繋げる。
不安を煽りすぎず、「自分のことだ」と感じてもらう共感設計を行う。

## Problem Section Direction

### Main Heading

最近、こんなお悩みありませんか？

### UI要素

- チェックリストカード
- 手書きメモ
- マーカー強調
- 不安そうな人物写真
- 下方向の矢印
- 「その原因は」リボン

### Checklist Examples

- 何から始めればいいか分からない
- 独学で練習しているけど上達しない
- 絵に自信がない
- 道具や衛生管理が不安
- 好きを仕事にできるか不安

### Handwritten Notes

- 独学で練習してるのに…
- 自分にもできる？
- 何が正解か分からない…

## Problem Layout Rules

- 白 / 生成り背景を基本にする
- 写真は右側または下部に配置
- チェックリストは左側または中央カード
- チェックアイコンは線画風
- 不安キーワードのみマーカーで強調
- 下部に次セクションへのリボン導線を置く

## Challenges Section Direction

### Main Heading

独学だけでは難しい世界です。

### 使用トーン

このセクションのみ、暗め背景を使用してよい。
ただし、黒を重くしすぎず、チャコール + ベージュ線で上品にまとめる。

### Topics

- 実践経験の不足
- 衛生管理の知識不足
- 肌質や個人差への対応
- タトゥー特有のデザイン力
- 動画やSNSだけでは限界がある
- 弟子入りのハードルが高い現実

## Challenges Layout Rules

- 縦並びの番号付きカード
- 各項目は `01〜06` の大きな番号を表示
- 左に番号、中央にテキスト、右に写真または図版
- 区切り線は細くする
- 写真は小さめの矩形で統一
- CTAではなく、次のFeaturesへ繋げる教育的導線にする

## Emotional Section NG

- 不安を過剰に煽るコピー
- 赤色の警告UI
- 真っ黒で重い背景
- 複雑なポスター構図
- 人物の表情が暗すぎる写真
- コーディング困難なコラージュ
