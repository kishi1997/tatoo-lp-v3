# 世界観

## LP全体のデザインコンセプト

Re'ink Tattoo School のLPは、以下の3要素を融合した世界観で統一する。

- 高級タトゥースタジオ
- 美容クリニックのような清潔感
- アートスクール / ギャラリーのような静けさ

ストリート感やハードなタトゥー感を前面に出すのではなく、未経験者でも安心して一歩を踏み出せる、上品で落ち着いた教育系LPとして設計する。

## 基本トーン

- 白ベース
- 余白を広く使う
- 黒は引き締め用に限定使用
- ベージュ / ゴールドを繊細なアクセントにする
- 写真はリアルなスタジオ・手元・人物中心
- 装飾は多いが、情報構造は常に整理する

## Typography Direction

### Japanese

見出しは上品な明朝体寄り、本文は読みやすいゴシック寄りを想定する。

推奨例：

- Heading: `Noto Serif JP`, `Yu Mincho`, `Shippori Mincho`
- Body: `Noto Sans JP`, `Yu Gothic`, `Hiragino Sans`
- English Accent: `Cormorant Garamond`, `Playfair Display`, `Cinzel`
- Handwritten Note: 筆記体風Webフォント、または画像/SVG装飾

## Layout Principles

- セクション幅は最大 `1080px〜1200px`
- モバイルLP想定では1カラムを基本にする
- PCでは2カラム / 3カラム / グリッドへ展開可能な構造にする
- 情報カードは実装しやすい矩形ベース
- 複雑な重なりは避ける
- 写真・テキスト・装飾を明確にレイヤー分けする
