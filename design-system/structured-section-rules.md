# Features/Curriculum

## 対象セクション

- Features Section
- Curriculum Timeline Section
- Instructor Section
- Works Gallery Section

## 目的

スクールの信頼性・教育内容・講師・実績を整理して伝える。
情緒的な装飾は残しつつ、構造は実装しやすいカード / グリッド / タイムラインで構成する。

---

# Features Section

## Main Structure

- セクションラベル
- 大見出し
- リード文
- 3つの特徴カード
- 下部の補足CTAまたは導線

## Features

01 少人数制の技術指導  
02 実践的なカリキュラム  
03 卒業後サポート

## Layout Rules

- 1特徴 = 1カード
- PCでは画像 + テキストの横並び
- SPでは縦積み
- 巨大な薄い番号を背景に配置
- 写真は角丸なし、または小さめ角丸
- カードは白背景 + 薄い影 + 細線
- 各カード間に小さな下向き矢印を置いてもよい

# Curriculum Timeline Section

## Main Heading

実践に特化した6回完結プログラム

## Steps

- STEP 01 道具の使い方
- STEP 02 ライン
- STEP 03 塗り
- STEP 04 ぼかし
- STEP 05 実践①
- STEP 06 実践②

## Layout Rules

- 縦タイムラインを基本にする
- 中央または左側に1本のtimeline line
- 各STEPは画像 + テキストの2カラム
- SPでは画像上、テキスト下の縦並び
- 大きな薄いステップ番号を背景に置く
- STEPラベルは小さなリボン型
- 各STEP間に薄い矢印を入れて流れを作る

# Instructor Section

## Layout Rules

- 白背景中心
- 講師写真 + プロフィールを横並び
- 下部にメッセージカード
- さらに3つ程度の補足写真 / キャプションを並べる
- 信頼性を最優先し、装飾は控えめ

## Include

- 講師名
- 肩書き
- 経歴
- 実績
- 得意スタイル
- メッセージ

## Tone

- calm
- trustworthy
- educational
- premium

---

# Works Gallery Section

## Layout Rules

- CSS Gridで再現できる規則的な画像配置
- 3カラムまたは2カラムのカードグリッド
- SPは1〜2カラム
- 画像下に短いキャプション
- タグを小さく右下に表示してもよい
- 不規則なmasonryは禁止

## Include

- タトゥー作品写真
- 施術中の手元
- スタジオ内観
- 道具 / インク / スケッチ

## Gallery Tone

- 明るい
- 余白がある
- 清潔感
- 作品が主役
- キャプションは控えめ

## Structured Section NG

- 複雑な斜めカード
- 過剰な重なり
- masonry風の高さバラバラすぎる配置
- 写真と文字の距離が近すぎるレイアウト
- 装飾が主役になる構成
