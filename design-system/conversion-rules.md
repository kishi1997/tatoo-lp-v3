# Price Section

## Main Heading

受講料金

## Price

398,000円（税込）

## Layout Rules

- 白背景ベース
- 中央に大きなpricing card
- 価格は大きく、余白を広く
- 料金に含まれるものをアイコン付きで整理
- サポート内容はチェックリスト化
- CTA前に軽い不安解消メッセージを置く

## Include

- 受講料金
- 全6回プログラム
- 税込表記
- 分割払い対応などの補足
- 料金に含まれるもの
- 安心のサポート体制
- LINE無料相談CTA

## Support Checklist Examples

- 少人数制だからこそできる、丁寧で細やかな指導
- 現場レベルの技術を、基礎から実践まで学べる
- 衛生管理・機材の扱い方まで、安全に配慮した指導
- 卒業後も相談できる、安心のアフターサポート
- 就職・独立の相談やポートフォリオ添削もサポート

## Price Decoration

- 価格上にリボンラベル
- `追加料金なし` の手書き吹き出し
- チェックアイコン
- 小さなスタジオ写真
- CTAへ向かう下向き矢印

# Final CTA Section

## Main Heading

好きな表現を、一生モノの仕事に。

## Sub Text

未経験からでも、あなたの挑戦を丁寧にサポートします。

## CTA

LINEで無料相談する

## Layout Rules

- 白背景 + 一部だけ淡い写真背景
- CTAエリアのみ少し濃い背景または大きなグリーンボタン
- 写真はスタジオ・手元・学んでいる人物
- ベネフィットをアイコンカードで3〜5個並べる
- 最後に短い感情コピーを置く

## CTA Button Rules

- LINEグリーンを使う
- 横幅は広く、押しやすく
- アイコン / テキスト / 矢印を含める
- ボタン周辺に十分な余白を取る
- CTAの直前に小さな誘導文を置く

## CTA Microcopy Examples

- まずは無料で相談してみませんか？
- 相談だけでもOK
- 無理な勧誘は一切ありません
- 簡単30秒で予約完了

## Conversion NG

- 赤や黄色で煽る価格表示
- カウントダウン風UI
- 限定感の過剰演出
- 情報商材っぽい装飾
- 価格よりCTAが先に目立ちすぎる構成
- 真っ黒で重いCTAセクション
