# 装飾

## 装飾の基本方針

装飾密度は高めでもよいが、情報の可読性を邪魔しないこと。
日本の美容LPらしい「視線誘導」「手書き感」「マーカー」「リボン」を使い、感情的な訴求と高級感を両立する。

## 使用してよい装飾

### 1. Ribbon Label

用途：セクションラベル、強調タイトル、CTA前の誘導。

例：

- `FEATURE`
- `CURRICULUM`
- `PRICE`
- `まずは無料で相談してみませんか？`

実装ルール：

- `border: 1px solid #CDBB9D`
- 背景は白 / 生成り / 黒の小面積
- 高さは控えめ
- 過度な立体感は禁止

### 2. Handwritten Notes

用途：不安・共感・補足・視線誘導。

例：

- `独学で練習してるのに…`
- `自分にもできる？`
- `まずはお気軽にご相談ください`
- `Your story starts here`

実装ルール：

- 文字色は薄いベージュ / グレー
- opacityを抑える
- 絶対配置は最小限
- モバイルでは非表示または小さくする

### 3. Marker Highlight

用途：重要キーワードの下敷き。

例：

- `一生モノ`
- `丁寧にサポート`
- `上達しない`
- `不安`

実装ルール：

- `linear-gradient(transparent 60%, rgba(181,154,106,.25) 60%)`
- 太すぎる蛍光色は禁止
- 1セクション内で2〜4箇所まで

### 4. Thin Divider Lines

用途：見出し下、カード間、グリッド区切り。

実装ルール：

- `1px solid rgba(181,154,106,.35)`
- 中央に小さなダイヤ装飾を置いてもよい
- 線は細く、余白を広く取る

### 5. Arrows / Eye Guidance

用途：下方向スクロール誘導、CTA誘導、ステップ進行。

実装ルール：

- 矢印は細線またはシェブロン
- 大きすぎる装飾矢印は禁止
- `↓`, `→`, `⌄`, SVG line arrow を使用
- 透明度を抑えて背景に馴染ませる

### 6. Faded Background Typography

用途：セクションの奥行き演出。

例：

- `ARTIST`
- `FEATURES`
- `WORRY`
- `CURRICULUM`

実装ルール：

- font-size: `80px〜180px`
- opacity: `0.03〜0.08`
- position: absolute
- テキストと重なっても読みにくくならない配置にする

## 禁止する装飾

- 過剰なグロー
- サイバーパンク風UI
- 映画ポスター的な強い光源
- 複雑すぎるコラージュ
- コーディング再現が難しい不規則な重なり
- 黒赤のハードなタトゥー表現
- ファンタジー / スピリチュアル表現
