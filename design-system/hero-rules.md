# FV

## 目的

FVでは、タトゥースクールの世界観とベネフィットを一瞬で伝える。
「趣味を仕事に変える」という感情的な期待感を、清潔で上品なタトゥースタジオの雰囲気と合わせて表現する。

## Main Copy

「趣味」を、一生モノの職へ。

## Sub Copy

最短3ヶ月で、タトゥーアーティストとしての一歩を。

## CTA

LINEで無料相談する

## Layout Rules

### PC

- 左：コピー / サブコピー / CTA
- 右：スタジオ写真または施術中の手元写真
- 背景には薄い大文字タイポグラフィ
- 左右を自然に繋ぐソフトな白グラデーションを入れる

### SP

- 上：写真
- 下：コピー / CTA
- 写真上にコピーを重ねすぎない
- CTAはファーストビュー内に必ず表示

## Hero Photography

使用する写真：

- タトゥーマシンを持つ手元
- 清潔なスタジオ内観
- タトゥーデザインを描いている様子
- 静かで落ち着いた人物写真

避ける写真：

- 露出が強すぎる写真
- 怖い印象のタトゥー写真
- 赤黒で攻撃的な写真
- クラブ / ストリート色が強い写真

## Hero Decoration

使用可能：

- 左上の縦リボン
- 薄い背景英字
- 細い罫線
- 手書き英字メモ
- 小さな矢印
- ゴールドのCTA枠

## CTA Design

- 背景は黒またはLINEグリーン
- 文字は白
- アイコン + テキスト + 矢印の構成
- 横幅は広め
- 角丸は控えめ、またはpill型
- hover時は少し明度を落とす

## NG

- 映画ポスターのような巨大人物構図
- 文字が写真に埋もれるレイアウト
- CTAが目立たない構成
- 黒背景が重すぎるFV
- 複雑な斜めレイアウト
